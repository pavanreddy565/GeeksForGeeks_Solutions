</strong><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">weight[]</strong><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">, you need to put these items in a knapsack of <p><span style="font-size: 14pt;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> values[] = [60, 100, 120], weights[] = [10, 20, 30], w = 50
<strong>Output: </strong>240.000000<strong>
Explanation: </strong>Take the item with value 60 and weight 10, value 100 and weight 20 and split the third item with value 120 and weight 30, to fit it into weight 20. so it becomes (120/30)*20=80, so the total value becomes 60+100+80.0=240.0 Thus, total maximum value of item we can have is 240.00 from the given capacity of sack. 
</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>values[] = [60, 100], weights[] = [10, 20], w = 50
<strong>Output: </strong>160.000000<strong>
Explanation: </strong>Take both the items completely, without breaking. Total maximum value of item we can have is 160.00 from the given capacity of sack.</span></pre>
<pre><span style="font-size: 14pt;"><strong style="font-size: 14pt;">Input: </strong><span style="font-size: 14pt;">values[] = [10, 20, 30], weights[] = [5, 10, 15], w = 100
</span><strong style="font-size: 14pt;">Output: </strong><span style="font-size: 14pt;">60.000000<br></span><strong style="font-size: 14pt; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Explanation: </strong><span style="font-family: -apple-system, system-ui, Segoe UI, Roboto, Oxygen, Ubuntu, Cantarell, Open Sans, Helvetica Neue, sans-serif;"><span style="font-size: 18.6667px;">In this case, the knapsack capacity exceeds the combined weight of all items (5 + 10 + 15 = 30). Therefore, we can take all items completely, yielding a total maximum value of 10 + 20 + 30 = 60.000000.</span></span><span style="font-size: 14pt;"><br></span></span></pre>
